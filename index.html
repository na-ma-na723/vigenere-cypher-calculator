VIGENERE CYPHER

Vigenere Cipher is a method of encrypting alphabetic text. It uses a simple form of polyalphabetic substitution. A polyalphabetic cipher is any cipher based on substitution, using multiple substitution alphabets. The encryption of the original text is done using the Vigenère square or Vigenère table.

  • The table consists of the alphabets written out 26 times in different rows, each alphabet shifted cyclically to the left compared to the previous alphabet, corresponding to the 26 possible Caesar Ciphers.
  • At different points in the encryption process, the cipher uses a different alphabet from one of the rows.
  • The alphabet used at each point depends on a repeating keyword.

Example :

Input :-
Plaintext : NAMANBHANDARI
Keyword : HELLO
Output :-
Ciphertext : UEXLBILLYRHVT
For generating key, the given keyword is repeated in a circular manner until it matches the length of the plain text.
The keyword "HELLO" generates the key "HELLOHELLOHEL" The plain text is then encrypted using the process explained below.

Encryption : Ei = (Pi + Ki) % 26
Decryption : Di = (Ei - Ki + 26) % 26

where
Pi is character of plain text at index i.
Ki is character of key at index i.
Ei is character of encrypted(cypher) text at index i.
Di is character of decrypted text at index i.
